# Employee Management System (Django + MySQL)

This is a web-based **Employee Management System** built using **Django, Python, and MySQL**.  
It allows users to manage employee information such as adding employees, assigning departments, viewing records, and updating or deleting details through a simple web interface.

This project is created for learning full-stack development using Django and database-driven applications.
